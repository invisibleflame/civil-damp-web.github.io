Enhancing Toughness and Modulus of Elasticity Properties of High Strength Concrete


Student:
Gurpreet Singh
Guide :
Prof. Prakash Nanthagopalan
Brief Description :
Developed fiber reinforced HSC using industrial by-products: micro silica, ultra-fine fly ash and slag. Performed tests of modulus of elasticity, compression, flexure and slump as per standard codes
Result/Conclusion :
Achieved 56 day strength of 103MPa by adopting Andreassen model to get maximum particle density
Future Scope :
In future you can work on the shrinkage properties of High Strength Concrete (HSC) or light weight HSC or can make HSC with the help of some other fibers like PVA fiber etc.
Prior knowledge :
None
Project Type :
In Sem Project
Software Used :
Elkem Materials Mixture Analyser (EMMA)
Research Type :
Lab work and Computational

Modelling and Analysis of Tensegrity bridge

Modelling and Analysis of Tensegrity bridge


Student:
Vivek Kumar
Guide :
Prof. Ian F.C. Smith
Brief Description :
The goals of the project were:
  • To model a 1:4 scale deployable tensegrity bridge using ABAQUS
  • Conduct damage detection tests and develop a Neural Network model for damage prediction

Result/Conclusion :
  • The displacement and the stress predictions were closer to the experimental values compared to the Dynamic Relaxation Model used earlier
  • No significant improvements could be achieved in the damage predictions but studies were continued.

Future Scope :
Structural Health Monitoring is a developing field with the convergence of Systems and Controls and Civil Engineering. Future studies could be in the are of damage location and self correction steps by the structure.

MATLAB, ABAQUS, Ansys
Research Type :
Mainly Simulations but had substantial lab work too.

Analysis of plates on elastic foundation

Analysis of plates on elastic foundation


Student:
Vivek Kumar
Guide :
Prof. Amit Das
Brief Description :
The project aims to the model and analyze the displacement and stress fields of plates on elastic foundations using Higher Order Theory
Result/Conclusion :
The displacement and the stress predictions were closer to the experimental values compared to the Poisson-Kirchhoff and Mindlin-Reisner Theories. There is a better determination of shear profile along the depth of the plate.
Future Scope :
Plates are ubiquitous and the better models are continuously developed for applications in Mechanical, Civil and Aerospace engineering. The future works could be aimed at using functionally graded plates or composite laminates.
Prior knowledge :
Advanced Solid Mechanics MATLAB Coding skills
Project Type :
B.Tech Project
Software Used :
MATLAB, FORTRAN
Research Type :
Theoretical
Miscellaneous :
The professor expects complete commitment and regular updates.

Risk Assessment in BOT Transportation Infrastructure

The goal of the project is to first identify the risks associated with a BOT project and then to find ways to mitigate those risks.
